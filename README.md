

# ShipSync 🚚

AI-Powered Dynamic Vehicle Routing & Delivery Optimization System

## 📌 Project Overview

ShipSync is an AI-powered logistics optimization platform designed to solve the Dynamic Vehicle Routing Problem (DVRP). The system continuously optimizes delivery routes based on live events such as traffic congestion, new delivery requests, cancellations, and vehicle availability.

## 🛠️ Tech Stack

- Frontend: React + Vite + Leaflet
- Backend: FastAPI
- Database: PostgreSQL
- AI: Genetic Algorithm + OR-Tools
- Maps: OpenStreetMap + OSRM
- Version Control: Git + GitHub
